## Commands
- `:Mason` to install LSP, lints and formatters
- `:WakaTimeApiKey` to put the API Key from [wakatime](https://wakatime.com/settings/api-key)
- `:Codeium Auth` to put the API Key from [codeium](https://codeium.com)
